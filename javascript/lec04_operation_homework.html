3. 연산자 총정리

여러분 연산자 배운다고 고생많았어요!
복습도 할겸 종합 문제를 풀어봅시다~
문제는 제가 드린 코드에 빈 script 태그를 채우는 거에요!!

1. 단항 연산자

버튼을 클릭하면 사용자가 입력한 숫자의 부호를
반대로 변경하여 alert창에 띄우세요. ※ HTML 태그를 보고 JS 코드를 유추해보는 문제입니다. ※

2. 논리 부정 연산자

참 radio를 클릭하면 false,
거짓 radio를 클릭하면 true값이
alert창에 출력되도록 코드를 추가하세요.

Hint. changeBool 함수의 매개변수로 논리형 변수를 사용하세요.

3. 증감연산자

num1은 20이고, num2는 0일때 버튼을 클릭했을때 나오는
num1과 num2의 값을 보고 다음중 어떤 연산이 이루어졌는지 선택하세요.

랜덤 버튼을 클릭하면 연산처리 후 결과값이 노란창에 출력됩니다.
이를 보고 전위 연산인지 후위 연산인지 판단하시어
아래 두개의 식 중에 하나를 선택하면 정답여부를 알 수 있습니다.

4. 복합대입연산자

버튼을 클릭하면 textarea안의 숫자에서 1만큼 빼기해서 textarea에 출력하세요.

Hint. 100을 10을 0을 하나 빼는게 아니라 100을 99로 1만큼 음수 연산하는 거에요~
